## Prompt From Tree (React)

Built-in ignore patterns: `*.env`, `.idea`, `__pycache__/`, `*.pyc`, `.DS_Store`, `.git`.

### Setup

```bash
npm install
```

### Run (frontend + server)

```bash
npm run dev:all
```

Then open `http://localhost:5173`.

### Run separately

```bash
npm run server
npm run dev
```
